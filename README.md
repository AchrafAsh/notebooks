## Stack

- `marimo` notebooks
- Keep it simple with `venv` and `pip`

## Setting up

Clone the repo
```bash
~ git clone https://github.com/AchrafAsh/notebooks.git
```

Create a virtual environment with Python3.10:
```bash
~ python3 -m venv env
```

Enable commands
```bash
~ source config
```

Install dependencies
```bash
~ roll install
```

Add packages you want
```bash
~ roll add ${package}
```
